【启动方式】

  方式一（推荐）：双击 "启动.bat"
  方式二：命令行执行 node server.js
  然后在浏览器中打开 http://localhost:3456

【系统要求】
  - Node.js 已安装（当前环境已具备）
  - 现代浏览器（Chrome / Edge 推荐）
  - 照片存储在本地磁盘

【使用流程】

  第一步：选片
    1. 点击顶部"📁 选择照片目录"
    2. 在弹出的对话框中选择照片根目录（如 D:\）
    3. 系统自动扫描并按年份/月份归类
    4. 左侧选择年份→月份浏览照片
    5. 点击照片选中（⭐精选），双击查看大图
    6. 使用左右箭头键快速浏览

  第二步：排版
    7. 选完照片后，点击右侧"去排版 →"
    8. 或点击顶部"✨ 自动排版"自动生成
    9. 每页可切换不同布局（单张/并排/四格等）
    10. 支持拖拽调整照片位置
    11. 可增删页面

  第三步：导出
    12. 切换到"预览"视图查看效果
    13. 点击"🖨️ 导出打印"
    14. 选择纸张尺寸和导出格式
    15. PDF 格式可直接带去打印店

【功能特点】

  ✓ 海量照片快速扫描（支持数万张）
  ✓ 按年份/月份自动归类导航
  ✓ 照片精选与搜索过滤
  ✓ 智能自动排版（8种布局模板）
  ✓ 拖拽调整照片位置
  ✓ 手动增删页面
  ✓ 多纸张尺寸支持（A4/A3/方形画册等）
  ✓ 导出高清 PDF（300dpi，适合打印）
  ✓ 导出单页 JPG 图片包
  ✓ 排版项目可保存/恢复（.album 文件）

【注意事项】

  - 首次扫描大量照片可能需要几十秒，请耐心等待
  - 导出 PDF 时每页需要渲染，大相册请耐心等待
  - 推荐使用 Chrome 浏览器获得最佳体验
  - 保存项目文件（.album）可以在不同时间继续编辑
